Prise en main

Votre Assistant Contextuel

ArchiAdmin est un assistant intelligent qui change de rôle selon le bouton activé. Il ne faut pas lui parler de la même manière si vous êtes en mode "Planning" ou en mode "Juridique".

Règle d'or : Sélectionnez toujours le bon mode dans la colonne de gauche avant de charger un fichier.

Bureau de Contrôle

Expert
Audit Réglementaire sur Plan

Le But

L'Assistant vérifie la conformité de votre plan avant envoi aux bureaux d'études. Il détecte les erreurs d'évacuation et de sécurité.

Les Entrées (Inputs)

  • Fichier Image du plan (JPG/PNG recommandé) chargée dans la zone "PLAN".
  • Paramètre Choix du règlement (ERP, Code Travail, Habitation).

L'Action

Cliquez sur un des 3 boutons : AUDIT GLOBAL, ÉVACUATION ou DÉSENFUMAGE.

Le Résultat

Un rapport technique sous forme de tableau. L'Assistant lit les cotes intérieures pour valider l'échelle et mesurer visuellement les culs-de-sac.

Urbanisme (PLU)

Extraction Réglementaire

Le But

Extraire les règles de gabarit d'un PDF de 300 pages en quelques secondes pour une parcelle donnée.

Les Entrées

  • Fichier PDF du règlement complet.
  • Texte Nom de la Zone (ex: UA, N).

L'Action

Cliquez sur LANCER.

Le Résultat

Un tableau synthétique : Retrait, Hauteur, Emprise au sol, Stationnement. L'Assistant cite la page du PDF pour vérification.

Comparatif & Marchés

Comparatif CCTP vs Devis

Le But

Vérifier si l'entreprise a bien chiffré tout ce qui est demandé dans le CCTP, et détecter les oublis.

Les Entrées

  • Source A Le CCTP (Cahier des Charges).
  • Source B Le Devis de l'entreprise.

L'Action

Cliquez sur LANCER.

Le Résultat

Un tableau comparatif ligne à ligne pointant les écarts, les oublis ou les surplus de prestations.

Rapport de Chantier

Compte-rendu Photo

Le But

Générer un CR de visite structuré par corps d'état à partir de notes en vrac et de photos.

Les Entrées

  • Photos Chargez vos photos en vrac.
  • Micro Dictez vos remarques : "Le peintre doit reprendre le salon, le maçon doit nettoyer."

L'Action

Cliquez sur LANCER.

Le Résultat

Un document PDF prêt à envoyer, où chaque remarque est classée par lot (Peinture, Maçonnerie) et associée aux photos.

Situation de Travaux

Validation Paiement

Le But

Valider le pourcentage d'avancement d'une entreprise pour débloquer son paiement.

Les Entrées

  • Source A Le Devis initial ou DQE.
  • Photos Photos de l'état actuel du chantier.

L'Action

Cliquez sur LANCER.

Le Résultat

L'Assistant estime visuellement l'avancement (ex: "Placo posé à 50%") et génère un tableau de validation.

Permis de Construire

Rédaction Notice PC4

Le But

Rédiger la notice architecturale obligatoire en décrivant le projet visuellement.

Les Entrées

  • Image Perspective 3D ou Façade.
  • Notes Petites précisions (ex: "Enduit ton pierre").

L'Action

Cliquez sur RÉDIGER NOTICE.

Le Résultat

Un texte juridique structuré (État initial, Insertion, Matériaux) prêt à être copié dans le Cerfa.

Juridique & Normes

Expertise Technique

Le But

Obtenir une réponse normative précise (DTU) ou rédiger un courrier formel (Mise en demeure).

Les Entrées

  • Texte Votre question ou le contexte du litige.

L'Action

Cliquez sur LANCER.

Le Résultat

Une réponse citant les articles de loi (CCAG Travaux) ou les normes NF-DTU en vigueur.

Outils Transverses

Envoyer

Fonction clé pour le mobile. Une fois un document généré, cliquez sur ce bouton pour l'envoyer immédiatement par Email, WhatsApp ou le sauvegarder sur Drive.

Export Excel

Ce bouton n'apparaît que si l'Assistant a généré un tableau (Planning, Devis, Audit). Il télécharge un fichier .csv exploitable.

Export PDF

Génère instantanément une mise en page A4 propre de tout ce qui est affiché à l'écran, prêt à être imprimé.
